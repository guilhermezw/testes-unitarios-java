
Este repositório foi criado para praticar e estudar testes unitários em Java.

O objetivo é aplicar conceitos como:

Estrutura de testes com JUnit 5
Uso de Mockito para criação de mocks
Testes de Services e regras de negócio
Boas práticas de organização e escrita de testes

Aqui serão implementados exemplos simples e cenários práticos para reforçar o aprendizado e evoluir na qualidade do código.

Tecnologias utilizadas
Java
JUnit 5
Mockito
Maven

Este projeto é exclusivamente para fins de estudo e prática.
